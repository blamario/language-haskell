# Revision history for language-haskell

## 0.1 -- unreleased

* Supports Haskell 2010 and a large number of GHC extensions
* Uses `deep-transformations >= 0.4.1` for `Full.Inward`/`Full.Outward` transformation modifiers,
  eliminating orphan `Full.Functor`/`Full.Foldable`/`Full.Traversable` instances
* Compatible with GHC 9.14.1
